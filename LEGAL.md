Last updated: 2026-07-15

This document describes the legal boundaries and intended use of **Q3 Arena + Team Arena Key Forge**. It is provided for transparency and is not legal advice.

## 1. Ownership requirement

Use this utility only if you lawfully own the relevant product:

- a legitimate physical or authorized digital copy of **Quake III Arena** for Arena use; and
- a legitimate copy of **Quake III: Team Arena** for mission-pack use, unless it is included with your edition.

Possession of this utility, its source code or a generated `q3key` file is not evidence of game ownership and does not create or transfer a game licence.

## 2. Open-source engine and non-free assets

id Software released Quake III Arena engine source code under the GNU General Public License. The source release exposes the local key-validation and file-handling routines documented by this proof of concept.

The GPL release applies to engine source code. It does not place the commercial game data under the GPL. In particular, this project grants no rights to Quake III Arena or Team Arena maps, textures, models, sounds, cinematics, `pak0.pk3` files, product keys, trademarks or other proprietary material.

The official ioquake3 documentation confirms that users must still purchase Quake III Arena and supply `pak0.pk3` from their own Steam, GOG or CD-ROM installation. Team Arena requires its corresponding mission-pack data.

## 3. Intended use

The utility is designed for:

- personal interoperability with a lawfully owned installation;
- recovery or replacement of a missing local `q3key` configuration file;
- preservation-oriented use with maintained community engines;
- personal play on community-operated servers whose rules allow the generated local file; and
- study of a file format and validation routine published in GPL source code.

The application is deliberately offline and writes only a user-selected 16-byte key file.

- avoiding the purchase of Quake III Arena or Team Arena;

- violating a server's rules, terms of service or key policy;
